# Multipage Digital Agency Website

A modern, high-performance, and fully responsive corporate website for a digital agency. This project demonstrates clean semantic layout, modern layout techniques, and interactive user experience implemented with vanilla JavaScript.

## 🔗 Live Demo
You can view the fully operational live website here:
👉 **[https://qq-astral666.github.io/digital-agency-portfolio/](https://qq-astral666.github.io/digital-agency-portfolio/)**

---

## 🛠️ Tech Stack & Methods
* **Markup:** Semantic HTML5 for better SEO and accessibility.
* **Styling:** Modern CSS3 featuring Flexbox, CSS Grid layouts, and native CSS Variables for easy scaling.
* **Architecture:** Component-based file structure.
* **Logic:** Pure Vanilla JavaScript (ES6+) without heavy external libraries or dependencies.
* **Responsiveness:** Mobile-first approach, fully optimized for all device sizes (Desktops, Tablets, Smartphones).

---

## 🚀 Key Features Implemented

1. **Multi-Page Architecture:** Smooth navigation across 5 interconnected semantic pages (Home, Services, Portfolio, About, Contacts).
2. **Dynamic Portfolio Filtering:** Instant grid item sorting by category (Web Development / Performance Marketing) handled directly via JavaScript.
3. **Interactive Lightbox Modal:** Clicking on any portfolio item opens an adaptive modal window dynamically loaded with specific project details.
4. **Animated Statistics Counter:** Numerical counters smoothly count up to their target metrics once scrolled into view using the `Intersection Observer API`.
5. **FAQ Accordion Component:** Elegant, custom-built accordion elements with smooth max-height transitions for frequently asked questions.
6. **Robust Form Validation:** Full client-side security verification for contact forms (RegEx email check, length validation) with instant error notifications.

---

## 📂 Project Structure
* `index.html` — Homepage with company introduction, numbers, and main service fields.
* `services.html` — Extended service descriptions and interactive FAQ accordion.
* `portfolio.html` — Case studies grid with live tab filters and modal popups.
* `about.html` — Corporate history section and team management showcase.
* `contacts.html` — Dual-column layout featuring contact info and verified feedback form.
* `style.css` — Global styling stylesheet organized with clear component comments.
* `script.js` — Core application logic and event handlers.
